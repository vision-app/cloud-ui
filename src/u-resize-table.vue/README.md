### Slots

#### (default)

插入`<u-resize-table-column>`子组件。


#### limit-text

在pattern属性值为limit时，数据的数目大于limit的值时默认在表格最后一行显示的提示内容

在pattern属性值为limit时，显示所有数据后默认在表格最后一行显示的提示内容

在列中`type`属性值为`'expand'`时，替换默认的切换图标
