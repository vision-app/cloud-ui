### 水平排布

``` html
<u-linear-layout direction="horizontal">
    <u-button>Button</u-button>
    <u-button>Button</u-button>
</u-linear-layout>
```

### 竖直排布

``` html
<u-linear-layout direction="vertical">
    <u-panel title="面板"></u-panel>
    <u-panel title="面板"></u-panel>
</u-linear-layout>
```

### 两端对齐

``` html
<u-linear-layout justify="space-between">
    <u-linear-layout>
        <u-button>Button</u-button>
        <u-button>Button</u-button>
    </u-linear-layout>
    <u-linear-layout>
        <u-button>Button</u-button>
        <u-button>Button</u-button>
        <u-button>Button</u-button>
    </u-linear-layout>
</u-linear-layout>
```
